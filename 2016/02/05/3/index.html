数学中从来不缺有意思的东西

2016-02-05
Math
发散级数

标题好正派！
确实是interesting thing…

我们定义一个函数
$$ \displaystyle \zeta(s) = \sum_{n=1}^\infty \frac{1}{n^s} = 1 + \frac{1}{2^s} + \frac{1}{3^s} + \ldots. \ \ \ \ \ (1) $$
当$ s=2 $时，有：
$$ \displaystyle \sum_{n=1}^\infty \frac{1}{n^2} = 1 + \frac{1}{4} + \frac{1}{9} + \ldots = \frac{\pi^2}{6}. \ \ \ \ \ (2) $$

然而当取 $ s=-1 $,则会发生有趣的事情：
$$ \displaystyle \sum_{n=1}^\infty n = 1 + 2 + 3 + \ldots = -1/12 \ \ \ \ \ (3) $$

还有更多。。。包括$\zeta(0)=-\frac{1}{2} $和$\zeta(-2)=0$等

其实它还有一般式！（惊恐状）
$$ \displaystyle \zeta(-s) = - \frac{B_{s+1}}{s+1} \ \ \ \ \ (4) $$
其中$ B_s $表示的是Bernoulli number,待我研究一下再说。。这个函数的名字叫做黎曼ζ函数

摘自陶哲轩博客［别告诉我不知道他是谁。。
