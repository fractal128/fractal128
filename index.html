Chapter 3 Mathematical basics

Fuzzy set: If G is a codomain, then the mapping u:G->[0,1]is a fuzzy set in G

{x|x in G, u(x) > 0} is called support of fuzzy set u
{x|x in G, u(x) = 0} is called kernel of fuzzy set u

Two fuzzy sets u1 and u2 are fuzzy-similar if they have the same kernel
Two fuzzy sets u1 and u2 are strongly fuzzy-similar, if they have the same kernel ans the same support

Significant changes are achieved by changing the kernel and the support of a fuzzy set.

{u(x)|x in G = constant for u(x): G ->[0,1]}

AND link: u1 AND u2: G->[0,1], with(u1 AND u2)(x) = MIN{u1(x), u2(x)}
OR link: u1 OR u2: G->[0,1], with(u1 OR u2)(x) = MAX{u1(x), u2(x)}

associative law

NOT: NOT do not have the same properties as the classical set theory.
u(A OR NOT A) != 1, u(A AND NOT A) != 0

Special operator: Modifiers
The modifiers can be used to adjust the linguistic values(fuzzy sets)
Leads to fuzzy-similar fuzzy sets.
concentration operation: CON{u(x)} = u(x) ^ 2
dilatation operation: DIL{u(x)} = sqrt(u(x))
contrast intensification: more focused at the bottom and less focused at the top

Fuzzy relation

Definition: Let G1 and G2 be two codomains and G1 G2 is the cross product(Cartesian product). Then R: G1 G2 -> [0,1] is a fuzzy relation links the codomains G1 and G2.

fuzzy relation matrix

fuzzy cross product: u1 u2 :G1 G2 ->[0,1] is (u1, u2)(x, y) = MIN{u1(x), u2(y)}

Fuzzy relation can be linked by multiply. * -> MIN, + -> MAX

MAX-MIN composition: mostly used

u2(y) = MAX(x in G1){MIN{u1(x), uR{x, y}}} = u1 \dot R

u2 is called fuzzy inference of u1 with respect to fuzzy relation R

Chapter 4 Fuzzy control

The number of fuzzy sets: between 2 and 7 are common

Fuzzification: determining the respective membership degree of the input signal for all fuzzy sets of the linguistic variable. —-> sympathy vector

Inference scheme

The level of cutting-off is given by the degre of fulfillment of the premise.

(1)One rule and one premise

(2)Several rules, one premise
The fuzzy of the active fuzzy rules are connected by OR

determine the active rules(the degree of fulfillment is larger than 0)

determine the fuzzy response for each active rules

Aggregation of the fuzzy response from all active rules

Defuzzification

(3)a single rule, several premises

A fuzzy rules with OR-linked premises can always be divided into several fuzzy rules with AND-linked premises when implementing fuzzy controller.

H:=MIN(u1(x1), u2(x2), …, un(xn)) is the degree of fulfillment of the rule’s premise.

(4)several rules, several premises

Defuzzification

(1)Maximum method(pattern recognition)

For the given numeric input signal, only the active fuzzy rule with the highest fulfillment counts

The fuzzy response of each rule is given as a singleton

The maximum method is particularly suitable for pattern recognition problems

(2)Maximum-Average method

if several rules have the same maximum degree of fulfillment, the the average of all corresponding numeric output signals is determined

interpolation between the singleton value is possible

(3)Maximum accumulation method

avoid overlap(Only one rule can be active)

(4)Center of gravity method

can’t cover all

(5)First of maximum method
